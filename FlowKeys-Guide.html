Setup Guide

Up and running in 5 minutes.

You'll download FlowKeys from GitHub — a website where developers share projects. You're simply downloading a folder. No account needed.

Step 1 — Download

1

Get FlowKeys from GitHub

  • Open github.com/rhalder90/FlowKeys in your browser
  • Click the green Code button
  • Click Download ZIP
  • Open your Downloads folder and unzip the file

Step 2 — Install Python

2

Make sure Python 3 is installed

FlowKeys is built with Python. Most Macs already have it.

Open Terminal (press Cmd + Space, type Terminal, press Enter) and run:

python3 --version

If you see a version number, you're good. If not, download Python from python.org/downloads

Step 3 — Install FlowKeys

3

Double-click install.command

Open the downloaded FlowKeys folder and double-click install.command. A Terminal window will open and handle everything automatically. Takes about 30 seconds.

Step 4 — Grant Permission

4

Allow Accessibility access (one-time)

The installer opens System Settings automatically. FlowKeys needs permission to detect your keypresses.

  • Click the + button
  • Press Cmd + Shift + H to go to your home folder
  • Open FlowKeys → select FlowKeys.app
  • Click Open
  • Make sure the toggle is ON (blue)

Tip: You only do this once. The permission survives restarts and updates.

Step 5 — Done

✓

Start typing

That's it. You should hear a mechanical click on every keypress. FlowKeys starts automatically every time you turn on your Mac.

Shortcuts

Two shortcuts. That's all.

Shortcut What it does
Cmd + Ctrl + K Turn sound on or off
Cmd + Ctrl + S Switch between mechanical and soft
Uninstall

Changed your mind? No worries.

Open Finder, press Cmd + Shift + H to go to your home folder, open the FlowKeys folder, and double-click uninstall.command.

Everything gets cleaned up. No leftover files.

Need Help?

We're here for you.

No sound when you type? The most common reason is the Accessibility permission from Step 4. Go back and make sure FlowKeys.app is added and toggled ON.
