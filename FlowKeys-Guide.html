Setup Guide

One command. That's all it takes.

1
Open Terminal (Cmd + Space, type Terminal, press Enter).
2
Paste this command and press Enter:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/rhalder90/FlowKeys/main/install-remote.sh)"
3
Grant Accessibility permission (one-time). The installer opens System Settings. Click + → press Cmd + Shift + H → open FlowKeys folder → select FlowKeys.app → click Open → toggle ON.
✓
Done. Start typing. FlowKeys auto-starts on every login. Nothing to remember.
Tip: You only grant Accessibility permission once. It survives restarts and updates.
Keyboard Shortcuts
Cmd + Ctrl + K

Cmd + Ctrl + S

Uninstall

Open Terminal and paste:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/rhalder90/FlowKeys/main/uninstall-remote.sh)"
